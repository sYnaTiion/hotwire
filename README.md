# Hotwire

Ein einfaches Hotwire Spiel mit React und Vite.

## Features
- Kabel per Drag & Drop verbinden
- Zeitlimit und Fehlversuche
- Runden- und Game-Over-Logik
- Visuelle Effekte für Erfolg und Fehler

## Starten

1. Abhängigkeiten installieren:
   ```
npm install
   ```
2. Entwicklung starten:
   ```
npm run dev
   ```

## Projektstruktur
- `src/components/CableGame.tsx` – Hauptspiel-Komponente
- `src/config/gameConfig.ts` – Spiel-Konfiguration
- `src/index.css` – Styles

## Konfiguration
Passe die Einstellungen in `src/config/gameConfig.ts` nach Bedarf an (z.B. Zeit, Kabelanzahl, Fehlversuche).
